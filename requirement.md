# Unilo 项目需求说明

## 1. 项目定位

Unilo 是一个面向小团队、朋友群体或兴趣社区的「私有协作空间」。它把实时聊天、异步动态、共享文件库和 AI 助手放在同一个工作台中，目标是让成员可以围绕话题、资料和历史上下文持续协作。

一期目标不是做完整的企业 IM 或网盘，而是完成一个可自部署、可多人使用、核心链路闭环的协作全家桶 MVP。

## 2. 一期 MVP 范围

### 2.1 必须完成

- 服务接入引导：用户首次进入前端页面/APP 时，需要输入后端服务地址和 secret key；验证通过后才进入登录/注册页面。验证成功后，前端保存后端服务地址用于后续 API/WebSocket 请求。
- 用户注册、登录、JWT 鉴权。
- 聊天频道：频道列表、创建频道、重命名频道、删除频道。
- 实时聊天：文本消息、Emoji、代码块展示、图片消息、消息回复、历史消息分页加载。
- 在线状态：基础 WebSocket 连接管理，支持新消息广播和频道列表变更广播。
- 异步动态：发布 Markdown 动态、动态列表、点赞/取消点赞、评论与二级回复。
- 共享资源库：目录浏览、创建文件夹、上传文件、下载文件、重命名、移动、删除。
- 文件预览：至少支持 `.txt`、`.md`、`.jpg`、`.png`、`.pdf`。
- AI 助手：可在频道中通过 @AI 或在独立 AI 面板发起对话；一期先支持读取站内上下文并返回答案。
- 本地开发基础设施：PostgreSQL、Redis、MinIO。

### 2.2 一期暂缓

- 私聊、群组权限、复杂组织架构。
- 端到端加密。
- 大规模音视频会议。
- 完整文件协同编辑。
- 完整插件市场。
- 复杂审批流、企业级审计后台。

### 2.3 一期验收标准

- 用户首次进入前端页面/APP 时，可以输入后端服务地址和 secret key 完成服务接入验证，验证成功后进入登录/注册页面。
- 新用户可以注册、登录并进入主界面。
- 多个浏览器窗口登录不同用户时，可以实时看到频道消息、频道变更和基础在线状态。
- 用户可以发布动态、点赞、评论，并看到自己的互动状态。
- 用户可以在共享文件库中上传文件、进入目录、预览常见格式、下载文件。
- 用户可以在频道中 @AI，AI 能基于频道最近消息或站内内容给出回答。
- 应用可以通过本地 docker compose 启动依赖服务并完成开发调试。

## 3. 产品模块

## 3.1 服务接入、认证与用户

### 功能

- 前端页面/APP 首次启动时展示服务接入页，要求用户输入后端服务地址和 secret key。
- 前端调用后端服务地址下的 `/api/v1/auth/verify` 校验 secret key。
- 验证通过后，前端进入登录/注册页面，并将后端服务地址保存到本地配置。
- 验证失败时，用户停留在服务接入页并提示重新输入。
- 用户通过用户名、密码、昵称注册。
- 用户通过用户名和密码登录，服务端返回 JWT。
- 登录态用于 REST API 和 WebSocket 鉴权。
- 用户资料包含昵称和头像地址。

### 规则

- `username` 全局唯一。
- 密码只保存哈希值，不能保存明文。
- secret key 只用于验证用户输入的后端服务是否可信；用户登录后仍以 Access Token + Refresh Token 作为业务鉴权凭证。
- 一期所有注册用户默认拥有相同权限。

## 3.2 聊天频道

### 功能

- 所有用户可以创建、重命名、删除频道。
- 频道列表对所有在线用户实时同步。
- 频道内支持：文本、Emoji、代码块、图片、文件类型消息。
- 支持回复某条消息。
- 支持向上滚动加载历史消息。

### 消息类型

- `text`：普通文本，允许包含 Emoji。
- `code`：代码块消息，`metadata.language` 可记录语言。
- `image`：图片消息，`metadata` 可记录宽高、文件 ID、缩略图地址。
- `file`：文件消息，`metadata` 可记录文件 ID、文件名、大小。
- `agent`：AI 助手消息。

### 规则

- 消息按服务端生成的递增 ID 排序。
- 历史消息使用游标分页，避免页码分页在新消息插入时错位。
- 删除频道时，一期可以软删除频道并隐藏其消息。

## 3.3 异步动态 Drops

### 功能

- 用户可以发布 Markdown 内容。
- 动态列表按发布时间倒序展示。
- 用户可以点赞/取消点赞。
- 用户可以评论动态，也可以回复评论。
- 列表中需要返回当前用户是否点赞。

### 规则

- `like_count` 作为冗余计数字段，用于列表快速展示。
- 点赞接口是 toggle 语义。
- 评论一期限制为两级展示：动态评论 + 回复评论；数据库可支持更深层级。

## 3.4 共享资源库 Workspace

### 功能

- 全体用户共享一个全局目录树。
- 支持创建文件夹、上传文件、下载文件、重命名、移动、删除。
- 支持拖拽上传和多选下载。
- 支持常见文件在线预览。
- 支持 `.md` 文件进入编辑模式并保存。

### 规则

- 文件与文件夹使用同一张表建模。
- `parent_id = null` 表示根目录。
- 上传前前端计算 SHA-256，服务端可通过哈希实现秒传。
- 对象内容存储在 MinIO 或兼容 S3 的对象存储中；数据库只保存元数据和 object key。
- 删除一期使用软删除并提供回收站，避免共享空间误删造成数据不可恢复。

## 3.5 AI 助手

### 使用场景

- 在频道中 @AI 提问。
- 在独立 AI 面板中提问。
- 让 AI 检索聊天记录、动态、文件名和可索引文本内容。
- 后续可扩展为通过自然语言触发站内动作。

### 一期能力

- 读取当前频道最近上下文。
- 搜索站内内容并总结。
- 以普通消息或 AI 面板消息形式返回结果。

### 后续能力

- 代表用户发布动态。
- 创建频道、整理文件、生成摘要。
- 调用第三方 API。
- 对上传文档建立全文索引或向量索引。

## 4. 推荐技术方案

### 4.1 前端

- Vue 3 + TypeScript + Vite。
- Tailwind CSS 或 UnoCSS 实现暗色、响应式 UI。
- TanStack Query for Vue 或 Pinia Colada 管理 REST 请求缓存。
- Pinia 管理轻量全局状态，例如当前用户、当前频道、WebSocket 状态。
- Milkdown 用于 Markdown 编辑与预览。
- Monaco Editor 或 Shiki 用于代码块高亮展示。

### 4.2 后端

- Go 1.22+。
- Gin 作为 HTTP 路由框架。
- PostgreSQL 作为主数据库。
- GORM 管理数据访问。
- 数据库迁移工具暂不决定；进入后端实现前再确认 goose、golang-migrate 或 GORM 迁移。
- Redis 用于 WebSocket 在线状态、Redis Pub/Sub 广播 fanout、短期缓存和未来任务队列。
- MinIO/S3 兼容对象存储保存文件内容。
- JWT 鉴权，采用 Access Token + Refresh Token 策略，中间件统一注入当前用户。
- WebSocket 使用 Gorilla WebSocket 或 nhooyr.io/websocket。

### 4.3 AI 与检索

- 一期使用 OpenAI-compatible API，由后端统一代理，避免前端持有模型 API Key。
- 一期使用 PostgreSQL 全文搜索和最近上下文拼接。
- 文件内容索引先支持 `.txt`、`.md`、可提取文本的 `.pdf`。
- 后续可以加入 pgvector 或独立向量库。

### 4.4 本地基础设施

- PostgreSQL：业务数据。
- Redis：实时状态与广播。
- MinIO：本地对象存储。

## 5. 页面与信息架构

### 5.1 首次接入流程

前端页面/APP 的入口流程：

1. 检查本地是否已有已验证的后端服务地址。
2. 如果没有，展示服务接入页，要求输入：
   - 后端服务地址，例如 `https://unilo.example.com` 或局域网地址。
   - secret key。
3. 调用 `${server_url}/api/v1/auth/verify`。
4. 验证成功后保存后端服务地址和服务信息，进入登录/注册页面。
5. 验证失败或服务不可达时，停留在服务接入页。
6. 用户后续可以在设置页切换后端服务，此操作应清空当前登录态。

### 5.2 桌面端主布局

- 左侧：空间入口、频道列表、Workspace 入口、Drops 入口。
- 中间：当前功能主视图，例如频道消息流、动态流、文件列表。
- 右侧：成员/动态摘要/AI 助手面板，可按页面折叠。
- 底部或局部输入框：频道消息输入、动态发布、AI 输入。

### 5.3 移动端主布局

- 底部 Tab：频道、动态、文件、AI、我的。
- 聊天频道内保持沉浸式消息列表 + 底部输入框。
- 文件库使用面包屑或顶部路径导航。

## 6. 权限模型

一期采用简单权限：开放注册，所有注册用户都可以读写所有频道、动态和共享文件。

后续可扩展：

- 空间管理员。
- 频道级权限。
- 文件夹级权限。
- 操作审计日志。

## 7. 待确认问题

- 数据库迁移工具选择 goose、golang-migrate 还是 GORM 迁移。
- OpenAI-compatible API 的具体供应商、模型和预算策略。
- 开放注册在公开部署时是否需要增加限流、验证码或后续切换为邀请码。
- 回收站中文件的自动清理周期。
